# Installation

## **1. Clone the Repository**

```bash
git clone https://github.com/untung29/broccoli-co.git
cd broccoli-co
```

## **2. Install Dependencies**

```bash
npm install
# or
yarn install
```

## **3. Run the development server**

```bash
npm run dev
```

## **4. Run the unit tests**

```bash
npm run test
```
